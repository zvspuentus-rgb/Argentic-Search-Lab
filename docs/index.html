Why this project matters

  • Runs locally on localhost with no mandatory subscription.
  • Fast search path for speed, deep path for quality.
  • Grounded research via MCP + URL context.

How to get best quality

  • LM Studio is recommended for local parallel requests.
  • Use stronger models for better synthesis/critique.
  • Use strict_repo_only=true for GitHub repo analysis.

What you can demo fast

  • Discovery to Deep Research flow.
  • MCP quick/deep tool calls.
  • Repo URL grounding with file-level context.

Quick Install

One-command bootstrap 
bash <(curl -fsSL https://raw.githubusercontent.com/zvspuentus-rgb/Argentic-Search-Lab/main/scripts/bootstrap.sh)
Manual install 
git clone https://github.com/zvspuentus-rgb/Argentic-Search-Lab.git
cd Argentic-Search-Lab
cp .env.example .env
docker compose up -d --build

Default endpoints: UI http://localhost:8093 · MCP http://localhost:8193/mcp · SearXNG http://localhost:8393/search?q=test&format=json
