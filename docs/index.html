Quick Install

One-command bootstrap 
bash <(curl -fsSL https://raw.githubusercontent.com/zvspuentus-rgb/Argentic-Search-Lab/main/scripts/bootstrap.sh)
Manual install 
git clone https://github.com/zvspuentus-rgb/Argentic-Search-Lab.git
cd Argentic-Search-Lab
cp .env.example .env
docker compose up -d --build

Default endpoints: UI http://localhost:8093 · MCP http://localhost:8193/mcp · SearXNG http://localhost:8393/search?q=test&format=json

FAQ

Is this really free to use?

Yes. The core stack runs locally with no mandatory subscription or per-query fee wall.

How is quality controlled?

Quality depends on model choice and mode. Deep Research with stronger local models yields better grounded answers.

When should I use strict_repo_only?

Use it whenever you analyze a specific GitHub repository and want only repo-scoped evidence.
